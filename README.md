# Entry Express React MySQL Boilerplate
A boilerplate with following features:
- Login / Sign up / Forgot Password - functionalities
- Backend
  - Express
  - BodyParser
  - CORS
  - formData (express-form-data)
  - Socket.io
- Frontend
  - React
  - React Router Dom
  - Sass/Scss
  - Socket.io-Client
  - Toast (custom)
